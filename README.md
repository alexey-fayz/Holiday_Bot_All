# Holiday Bot

Telegram-бот для работы с государственными и пользовательскими праздниками.

Проект разработан на Python и демонстрирует применение принципов объектно-ориентированного программирования, работы с API, обработки данных, асинхронного программирования и организации многомодульного приложения.

---

## Основные возможности

* Получение официальных праздников по дате
* Добавление пользовательских праздников
* Проверка и валидация дат
* Работа с локальным JSON-хранилищем
* Интеграция с внешним API праздников
* Асинхронная обработка запросов Telegram-бота
* Логирование и обработка исключений

---

### Модули

#### custom_holiday.py

Реализует класс `MyHolidays`, который расширяет библиотеку `holidays` и объединяет:

* государственные праздники;
* пользовательские праздники;
* загрузку данных;
* статистику вызовов;
* обработку исключений.

---

#### helper.py

Содержит вспомогательные функции:

* проверка формата дат;
* регулярные выражения;
* фильтрация списков дат;
* преобразование входных данных.

---

#### io_utils.py

Отвечает за безопасное чтение JSON-файлов через контекстные менеджеры.

---

#### handler.py

Модуль взаимодействия с внешним API праздников.

Поддерживает:

* HTTP-запросы;
* обработку ошибок сети;
* асинхронную загрузку данных.

---

#### holiday_launch.py

Точка входа приложения.

Содержит:

* запуск Telegram-бота;
* регистрацию команд;
* декораторы логирования;
* обработку пользовательских запросов.
